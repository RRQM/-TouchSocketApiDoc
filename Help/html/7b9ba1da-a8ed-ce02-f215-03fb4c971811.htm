      RpcMethod 属性

      属性

      HasByRef 是否有引用类型
      (继承自 Method。)
      HasCallContext 是否包含调用上下文
      HasReturn 是否具有返回值。当返回值为Task时，也会认为没有返回值。
      (继承自 Method。)
      Info 方法信息
      (继承自 Method。)
      IsEnable 是否可用
      Name 获取方法名
      (继承自 Method。)
      ParameterNames 参数名集合
      Parameters 参数集合
      ParameterTypes 参数类型集合，已处理out及ref，无参数时为空集合，
      ReturnType 返回值类型。

      当方法为void或task时，为null

      当方法为task泛型时，为泛型元素类型


      (继承自 Method。)
      RpcAttributes Rpc属性集合
      ServerFromType 注册类型
      ServerRpcAttributes Rpc服务属性集合
      ServerToType 实例类型
      TaskType 返回值的Task类型。
      (继承自 Method。)
      ToMethodInfo 实现方法。
