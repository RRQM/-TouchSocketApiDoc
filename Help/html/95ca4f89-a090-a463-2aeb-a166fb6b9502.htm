      INamedPipeServiceTClient 属性

      属性

      Config 设置项
      (继承自 IConfigObject。)
      Connected 用户连接完成
      Connecting 有用户连接的时候
      Count 当前在线客户端数量
      (继承自 INamedPipeServiceBase。)
      Disconnected 有用户断开连接
      Disconnecting 即将断开连接(仅主动断开时有效)。
      Logger 日志记录器
      (继承自 ILoggerObject。)
      MaxCount 获取最大可连接数
      (继承自 INamedPipeServiceBase。)
      Monitors 管道监听集合
      (继承自 INamedPipeServiceBase。)
      PluginManager 插件管理器
      (继承自 IPluginObject。)
      Resolver Resolver
      (继承自 IResolverObject。)
      ServerName 名称
      (继承自 IService。)
      ServerState 服务器状态
      (继承自 IService。)
      SocketClients 获取当前连接的所有客户端
      (继承自 INamedPipeServiceBase。)
