      Dispose 释放资源。内部已经处理了SuppressFinalize(Object)

      DisposeAsync 异步释放资源。内部已经处理了SuppressFinalize(Object)
      (继承自 DisposableObject。)
      EqualsDetermines whether the specified object is equal to the current object.
      (继承自 Object。)
      GetHashCodeServes as the default hash function.
      (继承自 Object。)
      GetTypeGets the Type of the current instance.
      (继承自 Object。)
      NoAutoClose 不处理Close报文。
      OnHttpRequest 在收到Http请求时。注意：此插件的执行在IHttpGetPlugin,IHttpPostPlugin, IHttpDeletePlugin,IHttpPutPlugin之前。
      OnTcpDisconnected 会话断开后触发
      OnTcpReceived 在收到数据时触发
      SetVerifyConnection(FuncIHttpSocketClient, HttpContext, Boolean) 验证连接
      SetVerifyConnection(FuncIHttpSocketClient, HttpContext, TaskBoolean) 验证连接
      SetWSUrl 用于WebSocket连接的路径，默认为“/ws”

      如果设置为null或空，则意味着所有的连接都将解释为WS

      ToStringReturns a string that represents the current object.

      UseAutoPong 当收到ping报文时，自动回应pong。
