      NATService 属性

      属性

      Config 设置项
      (继承自 SetupConfigObject。)
      Connected 用户连接完成
      (继承自 TcpServiceTClient。)
      Connecting 有用户连接的时候
      (继承自 TcpServiceTClient。)
      Count 当前在线客户端数量
      (继承自 TcpServiceBase。)
      Disconnected 有用户断开连接
      (继承自 TcpServiceTClient。)
      Disconnecting 即将断开连接(仅主动断开时有效)。
      (继承自 TcpServiceTClient。)
      DisposedValue 标识该对象是否已被释放
      (继承自 DisposableObject。)
      Logger 日志记录器
      (继承自 ConfigObject。)
      MaxCount 获取最大可连接数
      (继承自 TcpServiceTClient。)
      Monitors 网络监听集合
      (继承自 TcpServiceTClient。)
      PluginManager 插件管理器
      (继承自 SetupConfigObject。)
      Resolver Resolver
      (继承自 SetupConfigObject。)
      ServerName 名称
      (继承自 TcpServiceTClient。)
      ServerState 服务器状态
      (继承自 TcpServiceTClient。)
      SocketClients 获取当前连接的所有客户端
      (继承自 TcpServiceTClient。)
