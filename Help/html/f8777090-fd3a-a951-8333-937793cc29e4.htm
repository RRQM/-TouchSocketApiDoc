      IDmtpChannel 属性

      属性

      Available 具有可读数据的长度
      CacheCapacity 缓存容量
      CanMoveNext 判断当前通道能否调用MoveNext
      CanWrite 能否写入
      Id 通道Id
      LastOperationMes 最后一次操作时显示消息
      MaxSpeed 通道传输速度限制
      Metadata 元数据
      Status 状态
      TargetId 目的Id地址。仅当该通道由两个客户端打通时有效。
      Timeout 超时时间，默认1000*10ms。
      Using 是否被使用
