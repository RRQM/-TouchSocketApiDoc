      FileOperator 属性

      属性

      CompletedLength 已完成长度
      (继承自 FlowOperator。)
      FileSectionSize 文件分块大小，默认512*1024字节。 不要超过1024*1024字节。
      FlowGate 流量控制器。
      (继承自 FlowOperator。)
      IsEnd 由Result的结果，判断是否已结束操作。
      (继承自 FlowOperator。)
      Length 数据源的全部长度。
      (继承自 FlowOperator。)
      MaxSpeed 最大传输速度。
      (继承自 FlowOperator。)
      Metadata 元数据
      (继承自 FlowOperator。)
      Progress 进度
      (继承自 FlowOperator。)
      ResourceInfo 文件资源信息。此值不为空时复用，可能会尝试断点续传。
      ResourcePath 资源文件路径， 可输入绝对路径，也可以输入相对路径。
      Result 执行结果
      (继承自 FlowOperator。)
      SavePath 存放路径， 可输入绝对路径，也可以输入相对路径。 但是必须包含文件名及扩展名。
      Timeout 超时时间，默认10*1000ms。
      (继承自 FlowOperator。)
      Token 可取消令箭
      (继承自 FlowOperator。)
      TryCount 失败重试次数。默认10。
