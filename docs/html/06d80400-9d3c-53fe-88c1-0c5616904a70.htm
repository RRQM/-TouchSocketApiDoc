      ITcpJsonRpcClient 属性

      属性

      CanSend 表示对象能否顺利执行发送操作。

      由于高并发，当该值为True时，也不一定完全能执行。


      (继承自 ISenderBase。)
      CanSetDataHandlingAdapter 是否允许自由调用SetDataHandlingAdapter(SingleStreamDataHandlingAdapter)进行赋值。
      (继承自 IAdapterObject。)
      Config 设置项
      (继承自 IConfigObject。)
      Connected 成功连接到服务器
      (继承自 ITcpClient。)
      Connecting 准备连接的时候
      (继承自 ITcpClient。)
      DataHandlingAdapter 数据处理适配器
      (继承自 IAdapterObject。)
      Disconnected 断开连接
      (继承自 ITcpClientBase。)
      Disconnecting 即将断开连接(仅主动断开时有效)。


      (继承自 ITcpClientBase。)

      IP IP地址
      (继承自 ITcpClientBase。)
      IsClient 表示是否为客户端。
      (继承自 ITcpClientBase。)
      LastReceivedTime 最后一次接收到数据的时间
      (继承自 IClient。)
      LastSendTime 最后一次发送数据的时间
      (继承自 IClient。)
      Logger 日志记录器
      (继承自 ILoggerObject。)
      MainSocket 主通信器
      (继承自 ITcpClientBase。)
      Online 判断是否在线
      (继承自 IOnlineClient。)
      PluginManager 插件管理器
      (继承自 IPluginObject。)
      Port 端口号
      (继承自 ITcpClientBase。)
      Protocol 终端协议
      (继承自 IClient。)
      RemoteIPHost 远程IPHost。
      (继承自 ITcpClient。)
      Resolver Resolver
      (继承自 IResolverObject。)
      UseSsl 使用Ssl加密
      (继承自 ITcpClientBase。)
