      IIdRequsetInfoSender 方法

      方法

      Send 同步发送数据。

      内部已经封装Ssl和发送长度检测，即：调用完成即表示数据全部发送完毕。

      该发送会经过适配器封装，具体封装内容由适配器决定。

      SendAsync 异步发送数据。

      在ITcpClient时，如果使用独立线程发送，则不会触发异常。

      在ITcpClientBase时，相当于BeginSend(Byte, Int32, Int32, SocketFlags, SocketError, AsyncCallback, Object)。

      该发送会经过适配器封装，具体封装内容由适配器决定。
