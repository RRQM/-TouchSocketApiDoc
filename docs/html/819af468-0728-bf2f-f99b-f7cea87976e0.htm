      SetNamedPipeDataHandlingAdapter 设置(命名管道系)数据处理适配器。
      SetNamedPipeListenOptions 直接单个配置命名管道监听的地址组。
      SetPipeName 当管道在本机时，仅设置管道名称即可。
      SetPipeServer 设置命名管道的主机名称。
      UseNamedPipeReconnection(IPluginManager, TimeSpan, FuncINamedPipeClient, Int32, Exception, Boolean, ActionINamedPipeClient) 使用命名管道断线重连。

      该效果仅客户端在完成首次连接，且为被动断开时有效。

      NamedPipeDataHandlingAdapterProperty 数据处理适配器，默认为获取NormalDataHandlingAdapter 所需类型FuncTResult
