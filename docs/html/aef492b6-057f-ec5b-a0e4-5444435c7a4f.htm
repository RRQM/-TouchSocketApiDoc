      NamedPipeSocketClient 属性

      属性

      CanSend 表示对象能否顺利执行发送操作。

      由于高并发，当该值为True时，也不一定完全能执行。

      CanSetDataHandlingAdapter 是否允许自由调用SetDataHandlingAdapter(SingleStreamDataHandlingAdapter)进行赋值。
      Config 设置项
      (重写 ConfigObjectConfig)
      DataHandlingAdapter 数据处理适配器
      Disconnected 断开连接
      Disconnecting 即将断开连接(仅主动断开时有效)。

      DisposedValue 标识该对象是否已被释放
      (继承自 DisposableObject。)
      Id 服务器唯一Id。
      IsClient 表示是否为客户端。
      LastReceivedTime 最后一次接收到数据的时间
      LastSendTime 最后一次发送数据的时间
      Logger 日志记录器
      (继承自 ConfigObject。)
      Online 判断是否在线

      该属性仅表示管道状态是否在线

      PipeStream 用于通讯的管道流。
      PluginManager 插件管理器
      Protocol 终端协议
      Resolver Resolver
      Service
